# 📖 Dictionary Bot – Ваш личный помощник в изучении слов!

**Dictionary Bot** – это удобный Telegram-бот, который поможет вам эффективно запоминать иностранные слова и термины, организовывать их по папкам и тренировать память с помощью различных игровых режимов! 🚀

---

## 🔥 Основные возможности

### 📂 Организация словаря
✅ **Папки** – создавайте неограниченное количество тематических коллекций слов.  
✅ **Добавление слов** – вручную или с импортом из файлов **TXT/DOCX**.  
✅ **Гибкие разделители** – сами выбирайте символы для разделения терминов и значений!

Пример формата:
```plaintext
apple: яблоко
car: машина
```
_(разделители можно настроить в настройках)_

---

### 🎮 Игровые режимы
| 🎯 Режим | 🔍 Описание |
|----------|------------|
| 🧠 **Запоминание** | Карточки с кнопками **"Помню"** и **"Не помню"**, возможностью показа ответа. |
| ✍️ **Тестирование** | Ввод ответа вручную с автоматической проверкой правильности. |

**🔄 Гибкость:** Можно менять местами термины и значения, проверяя себя в обоих направлениях!

---

### ⏰ Напоминания
💡 Бот может отправлять напоминания о повторении слов! Вы сами выбираете **дни недели и время**, когда вам удобно получать уведомления. Не забудете ни одного слова! 😉

## 🚀 Запуск проекта

Для запуска необходима Java 17+.

### 🔹 1. Клонирование репозитория
```bash
git clone https://github.com/nerlisse/dictionaryBot.git
```

### 🔹 2. Настройка конфигурации
Создайте или отредактируйте `src/main/resources/application.properties`:
```properties
bot.token=ВАШ_ТОКЕН_БОТА
spring.datasource.url=jdbc:mysql://localhost:3306/dictionary
spring.datasource.username=user
spring.datasource.password=pass
```

### 🔹 3. Запуск бота
```bash
./mvnw spring-boot:run
```
